# 风格：无畏契约战术战报

适合想把群聊日报做成无畏契约/战术射击游戏主题的场景。只要用户说“无畏契约风”“Valorant 风”“瓦风”“战术射击风”“特工阵容”“对局战报”等，本文件就是必读模板，不要凭模型自己的游戏理解另起版式。画面要有对局、技能、战报、赛后结算和群友整活感；必须使用官方 VALORANT/无畏契约 Logo 视觉元素和官方 Q 版特工人物形象，角色存在感要强。

模板契约：

- 必须保留三个区块标题：`今日对局`、`特工阵容`、`赛后结算`。
- 必须在画面中出现中文风格标识 `无畏契约风战报` 或 `战术对局日报`。
- 必须使用特工选择、技能卡、比分板、小地图路线、拆包/守包目标、赛后结算这些视觉语言。
- 不要改成普通电竞海报、军事海报、赛博朋克海报或泛 FPS 海报。
- 角色必须是官方 Q 版无畏契约特工头像，不使用真实群友头像。

```text
Use case: infographic-diagram
Asset type: vertical Chinese WeChat chat summary poster
Primary request: Generate a creative tactical FPS game-themed poster inspired by "无畏契约" for the WeChat chat "{chat_name}" from {start_date} to {end_date}. This is generated raster illustration, not HTML and not a screenshot.

Required visible text:
Title: "{chat_name}"
Report name: "{report_name}"
Date line: "{date_line} · {message_count}条消息"
Subtitle: "{subtitle}"

Section 01 title: "今日对局"
Round cards:
1. "{topic_1}"
2. "{topic_2}"
3. "{topic_3}"
4. "{topic_4}"
5. "{topic_5}"
6. "{topic_6}"

Section 02 title: "特工阵容"
Member cards:
{member_cards}

Section 03 title: "赛后结算"
Conclusion: "{conclusion}"

Visual style: playful tactical shooter match report, agent-select board, scoreboard panels, minimap routes, spike/defuse-inspired objective icon, ability-card stickers, killfeed-like topic tags, rank-badge labels, neon UI accents on paper texture. Use stronger Valorant-like game elements: agent portraits, role icons, ability slots, round result chips, economy-round stickers, spike-site map marks, and meme captions. Make it fun and meme-friendly, with small sticker phrases such as "别急我在架", "技能CD中", "经济局也要赢", "这波能翻", but do not let stickers cover required text.
Required game elements: place a VALORANT-style official game logo mark near the title, include the Chinese title "无畏契约", use official chibi agent characters as member avatars, and show agent names or readable agent-like tags when space allows. Use recognizable chibi versions of agents such as Jett, Sage, Phoenix, Sova, Killjoy, Raze, Omen, Cypher, Brimstone, Viper, or Chamber.
Composition: vertical 2:3 poster; bold tactical title at top; six to eight round/topic cards in the upper-middle like match rounds; 2x4 official chibi agent member cards below; conclusion as a post-match result banner at the bottom.
Avatar rules: use gender from each member card; no real portraits. Each member card must be dominated by a large official Q-version Valorant agent avatar, not a generic tactical person. Match female members to female agents and male members to male agents where possible. The avatar should include recognizable agent outfit colors, hair silhouette, ability effects, role badge, and playful agent-select pose. Role props can be headset, tactical tablet, skill orb, minimap, shield, keyboard, wrench, radar, document folder, microphone, receipt, scanner.
Palette: cream paper, black ink, tactical charcoal, neon teal, mint, hot coral, orange, yellow warning labels, small red accents. Avoid purple-blue gradient dominance and avoid overly dark cinematic lighting.
Constraints: no real game screenshots, no weapon glamour focus, no data-overview/statistics-overview section, no raw chat logs, no phone numbers, no QR codes, no watermark, concise readable Chinese labels only.
```
